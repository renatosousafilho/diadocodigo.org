Sobre

O Dia do Código é um evento feito por programadores para programdores. O objetivo do Dia do Código é realizar encontros onde desenvolvedores possam se reunir para falar sobre suas experiências com Linguagens de Programação e Desenvolvimento de Software em Geral. Desde Desenvolimento de Games, Aplicações Web até gerenciamento de Servidores Web.

Este evento só é possível devido a união do Teresina Hacker Club(http://teresinahc.org), GURUPI(Grupo de usuários Ruby) e PUG-PI(Grupo de Usuários Python) mas o foco é fomentar o Encontro de Programadores sem restrições.

O evento é totalmente Gratuito e qualquer pessoa que se caracterizar como no mínimo Curioso a respeito do mundo do Código está apto a participar para assistir apresentações com profissionais da área.

Let`s code!

Agenda

Time Slot Description
15h00 Stacks de Softwares. Com tantas linguagens, frameworks, técnicas, plataformas, bancos de dados e mais um universo de coisas, o que usar para se ter uma boa, robusta aplicação? Veja o que são Software Stacks, como arquitetar uma e referencias sem graça a filmes antigos
16h00 Dissecando o Django Admin. É comum notar aspirantes ao desenvolvimento de aplicações Web reclamando do alto nível da curva de aprendizado de algumas linguagens de programação ou mesmo de alguns frameworks. O Django é um framework que possui no seu arcabouço uma API para desenvolvimento ágil de formulários de cadastro e listas com uma fácil integração a Banco de Dados seguindo a arquitetura MVC. O objetivo dessa palestra é apresentar o Django Admin e suas peculiaridades e explicar por que ele pode ser uma boa opção para novos programadores e mesmo para programadores veteranos.
17h00 Deploy de Aplicações com Capistrano. Meio Norte Desenvolver uma aplicação é um processo árduo, mas garantir que sua aplicação vai subir para um ambiente de produção pode se tornar um pesadelo maior ainda. Nesta apresentação descreverei um pouco a respeito do Capistrano, uma biblioteca Ruby que facilita o processo de Deploy de Apps Web, não só feitas com Ruby on Rails mas com outros frameworks
